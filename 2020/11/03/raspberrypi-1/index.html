RasperryPi install hass.io and more

Created2020-11-03|Updated2020-11-03
|Word count:591|Reading time:2min|Post View:
  1. Format tool
  2. Raspbian Buster with desktop https://www.raspberrypi.org/downloads/raspbian/
  3. download balenaEtcher https://www.balena.io/etcher/
  4. flash system to TF card then will show two disks in your file-explorer
  5. open the disk named boot , establish on empty SSH file
  6. 把TF卡插入树莓派4，然后启动。查看路由器中新增的设备，通过SSH软件进行访问，默认用户名密码为：pi/raspberry,进入系统。 用PUTTY
  7. 输入sudo raspi-config,进入如下界面。 选择4 Localisation Options
  8. 选择I2 Change Timezone->Asia->Shanghai，
  9. Resolution 改成第三个 为了VNC能显示
  10. VNC功能打开
  11. 退出（自动重启）
  12. 安装docker-ce
    输入命令：sudo -i
                  apt-get install software-properties-common
                  apt-get update
                  apt-get install -y apparmor-utils apt-transport-https avahi-daemon ca-certificates curl dbus jq network-manager socat
    curl -fsSL get.docker.com | sh
    其中，最后一步安装时间挺长的，请耐心等待 （亲测一个小时左右）（指令一条一条来，可以重复几遍，看成功没有）
  13. 12、安装hassio
    curl -sL “https://raw.githubusercontent.com/home-assistant/hassio-installer/master/hassio_install.sh" | bash -s – -m raspberrypi4
    （时间比较长，一直等到出现 [info]RUN HASS.IO 为止
  14. 安装完成
  15. 打开raspberry ip:8123 例如 192.168.1.13:8123 可能当时无法立即打开，等一会，估计十几分钟。
  16. 一直到能打开8123且看到蓝色房子，第一次需要十几二十分钟来初始化。
  17. 开启samba 电脑局域网访问树莓派文件。
  18. sudo apt-get install samba
    编辑samba的配置文件，把只读改改成可读写等，命令：sudo nano /etc/samba/smb.conf
    将配置文件中的 read only =no
    增加用户pi并设置密码：sudo smbpasswd -a pi
    重启samba服务：sudo systemctl restart smbd
  19. 电脑CMD输入，映射到电脑Y盘 net use y:  \树莓派IP地址\pi 登录密码  /user:pi
    https://bbs.hassbian.com/thread-8044-1-1.html 添加configuration tool 可以直接在界面改configuration.Yaml
  20. 树莓派连接蓝牙设备 手机下载 蓝牙调试器 查看家里蓝牙设备MAC地址
  21. For Example : connect 88:2D:53:11:C3:D0 trust 88:2D:53:11:C3:D0
  22. 设置 自启动 sudo crontab -e @reboot sleep 15 && echo “connect 88:2D:53:11:C3:D0” | bluetoothctl

以上是纯净系统安装hass.io的老方法，比较容易出问题且网速无法保证。
目前采用最简易的办法，就是下载hachina上装有hass.io的镜像就可以了，省去了很多的麻烦。
https://www.hachina.io/docs/23131.html
